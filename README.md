**project overview**

This project is a simple Login Form built using HTML,CSS, and JS

It demonstrates client side form validation for email and password fields with real-TIME ERROR messages.

**Technologies Used**

1.HTML- Structure of the login form

2.CSS - Styling and layout

3.JS - Form valiadtion logic
